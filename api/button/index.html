Button(config)

config

类型：Object

描述：配置项

名称类型描述可选
checkableString按钮是 toggle button否
contentString按钮的显示内容否
describedbyString按钮节点的 aria-describedby 属性值是
tooltipString按钮节点的 title 属性值是
valueString按钮值是
