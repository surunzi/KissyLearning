Base

功能创建类
模块名base
模块实质类
父类Attribute
备注请使用 Base.extend() 创建类，而不是 new Base()

构造函数

Base(config)

config

类型：Object

描述：属性名/配置信息对

名称类型描述
listenersObject配置组件的事件绑定
pluginsArray插件构造器数组或插件对象数组
describedbyString按钮节点的 aria-describedby 属性值
tooltipString按钮节点的 title 属性值
valueString按钮值

静态方法

.extend(extensions, methodDesc, staticAttributes, desc)：从当前类上扩展出一个子类

extensions（可选）

类型：Array

描述：扩展类数组

methodDesc

类型：Object

描述：方法集合键值对

staticAttributes

类型：Object

描述：放到新产生组件类上的静态属性集合键值对，其中 ATTRS 属性特殊对待

desc

类型：Object

描述：类元信息

方法

.callSuper()：调用父类的对应方法，如果没有，则返回undefined

.destroy()：销毁实例

.getPlugin(id)：根据指定的 id 获取对应的plugin实例

id

类型：String

描述：plugin实例的id

返回值

描述：对应的plugin实例

.plug(plugin)：安装指定插件

plugin

类型：Function | Object

描述：指定的插件构造器或者插件对象

.unplug(plugin)：卸载指定插件

plugin

类型：Function | Object

描述：指定的插件 id 字符串或者插件对象
