Xtemplate

功能富逻辑的 KISSY 模板引擎，兼容 mustache
模块名xtemplate
模块实质类

构造函数

XTemplate(tpl, config)

tpl

类型：String | Function

描述：字符串模板或编译过的模板函数

config

类型：Object

描述：配置项

名称类型描述
cacheBoolean是否缓存生成的模板函数，默认 true
commandsObject局部自定义命令的键值对
nameObject模版名称，可在 chrome dev tools 中看到以该值为名称的模板文件代码

静态方法

.addCommand(commandName, fn)：添加全局命令

commandName

类型：String

描述：命令名称

fn

类型：Function

描述：命令定义

方法

.addCommand(commandName, fn)：添加局部命令

commandName

类型：String

描述：命令名称

fn

类型：Function

描述：命令定义

.compile(subTplName)：编译模板

subTplName

类型：String

描述：子模板名称

返回值

类型：Function

描述：编译过的模板函数

.removeCommand(commandName)：删除指定的局部命令

commandName

类型：String

描述：命令名称

.render(data)：渲染数据到模板

data

类型：Object

描述：数据对象

返回值

类型：String

描述：数据融合模版后的字符串
